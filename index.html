Hi there, I'm Tracy!

I'm a program manager/maker/computer scientist/artist.

As a Microsoft program manager, I build Modern Input features such asink to math and a PowerPoint presenter coach that promotes inclusive presentations.
As a researcher, I fabricated tactile interfaces for mobile accessibility and simulated the corrosion of nuclear waste canisters.
I create interactive art ranging from a whimsical vending machine that trains AI systems to maternity shirts that respond to inappropriate touching.

I'm intrigued by exploring human connections to each other and our environments.
